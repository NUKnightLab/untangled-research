Tools

Collecting data. Storing information. Finding connections. Displaying those findings. We've gathered some notable tools that facilitate the proccess of network analysis in journalism.

Neo4J

Neo4J is a graph database that allows users to "ask and answer" data questions. It is a databse that allows you to model, store, and query data in the form of a graph with nodes and relationships that you can attach properties to. After structuring your graph you can query it. The data modeling experience aims to be simple and attempts to let users build a graph model that looks like the questions you want to ask of your domain.
